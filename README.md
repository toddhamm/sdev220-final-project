Client
An attendance coordinator for a school corporation

Scope
The program will accept a csv file consisting of raw attendance data for all students in a grade
level at a local school. It will parse the data and output a report in csv, PDF and on screen
showing a pattern of five or more absences in one or more categories (total days absent, total
mornings only absent, total afternoons only absent) as well as a summary of any student who
has five or more unexcused absences.

Purpose
The school corporation is lacking a report of this type that shows trends in absences. They are
currently creating a report similar to this by hand. This will save them time as well as illustrate
some trends in absences
